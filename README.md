# Hello World Node App

A simple Node.js application that prints "Hello, World!".

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed

### Installation

```bash
git clone https://github.com/yourusername/helloworldnodeapp.git
cd helloworldnodeapp
npm install
```

### Usage

```bash
node index.js
```
